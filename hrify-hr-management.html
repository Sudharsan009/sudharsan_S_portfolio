HRiFY – HR Management App

A mobile-first HR solution designed to simplify employee and workforce management. Built with Flutter and Firebase, this comprehensive application streamlines HR operations and reduces administrative workload by 40%.

The application features core modules including employee management, attendance tracking, leave requests, and payroll automation. With a secure role-based access system, it protects sensitive employee data while providing an intuitive dashboard for quick access to critical HR metrics.

Key Features:

  • Employee management and profile system
  • Automated attendance tracking and reporting
  • Leave request management and approval workflow
  • Payroll automation and salary calculations
  • Role-based access control for data security
  • Mobile self-service features for employees
  • Real-time HR metrics and analytics dashboard
