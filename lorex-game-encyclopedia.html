LoreX – Game Lore Encyclopedia App

A mobile encyclopedia app for exploring and contributing to video game lore. Designed and developed with structured content viewing, spoiler-aware features, and interactive relationship graphs to create an immersive gaming knowledge platform.

The application implements Firebase Auth with custom user roles (Contributors, Moderators) to enable community-driven content submission and moderation. Built with a dynamic timeline system to organize lore events chronologically across franchises, with gamification features to boost user engagement.

Key Features:

  • Structured content viewing with spoiler protection
  • Interactive relationship graphs for character connections
  • Community-driven content submission system
  • Dynamic timeline for chronological lore organization
  • Gamification with XP, badges, and user progression
  • Role-based moderation system
  • Scalable architecture for franchise expansion
